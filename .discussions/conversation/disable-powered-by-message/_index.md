### Summary

This guide explains how to disable the "Powered by" message displayed in the conversation when using the AI Agent Messenger SDK.

### Guide & Snippet

### Contact Delight AI Support to Change the Configuration

If you want to activate Powered by message, you can request a backend configuration change.

To proceed with this method:

1. Visit: [Delight AI Agent support](https://sendbird.com/support)
2. Submit a request asking to **disable the automatic Powered by message** for your application
3. Our team will review and apply the configuration change as needed
